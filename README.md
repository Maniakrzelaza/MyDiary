# MyDiary
ASP.NET Core MVC Diary with tests

## Running application: 

`dotnet run`

## Running unit tests

`dotnet test`


## Architecture
- MVC
- ViewModel pattern
- Dependency Injection
- Entity - Model mapping

## Frameworks and tools
- ASP.NET Core MVC
- ORM: EntityFramework Core

## Tests
- NUnit with Moq
